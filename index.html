How to use

Search %s result %n return %m

Search %s="Query" result %n="What result? 1 first, 8 last" return %m="Title, URL, Content"

Searches the internet using google

How it works?

http://ajax.googleapis.com/ajax/services/search/web?v=1.0&q=(query)&rsz=1&start=(what result)&safe=active
